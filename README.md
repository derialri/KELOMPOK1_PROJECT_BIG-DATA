## 📝 Deskripsi Proyek
Seluruh proses pengolahan data—mulai dari tahap ekstraksi, pembersihan data (data cleaning), hingga visualisasi hasil analisis—kami rangkum ke dalam satu file Google Colab (`.ipynb`) yang ada di repositori ini. 

Karena dataset yang kami gunakan cukup besar (lebih dari 500 MB), kami merancang agar program ini efisien dan praktis saat diuji.

## Cara Menjalankan Program
Program ini sudah kami atur sedemikian rupa agar yang melihat repositori kami tidak perlu repot mengunduh atau mengunggah dataset secara manual. 

Untuk melihat hasilnya:
1. Buka file notebook `.ipynb` kami melalui Google Colab.
2. Klik menu **Runtime** di bagian atas, lalu pilih **Run all**.
3. Selesai! Sel di baris paling awal akan otomatis mengunduh dataset secara aman dari *cloud* kami, lalu mengeksekusi seluruh analisis dan grafiknya hingga akhir tanpa terjadi *error* pada direktori data.
